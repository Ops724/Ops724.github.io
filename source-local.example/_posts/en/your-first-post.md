This is a starter template for your first English post.

Your first post does not need to be perfect. It only needs to help you begin writing in a way you can sustain.

<!-- more -->

## What you can write about

Pick one simple direction:

1. why you started this blog
2. what you have been learning lately
3. what you want this space to hold over time

## A simple structure

Three short sections are enough:

1. an opening
2. one thing that has been on your mind
3. one reason you want to keep writing here

## If this is a technical post

You can also start with a very small example:

```js
function greet(name) {
  return `Hello, ${name}`;
}
```

Close with one thread you may continue in the next post.
