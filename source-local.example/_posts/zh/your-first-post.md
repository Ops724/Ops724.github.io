这是你的第一篇中文文章模板。

你可以先用它写一篇很轻的开场内容，不一定要正式，也不必太长。第一篇文章更重要的是帮你把写作节奏建立起来。

<!-- more -->

## 可以写些什么

你可以从下面这些方向里选一个：

1. 为什么想重新开始写博客
2. 最近在想什么、在学什么
3. 你希望这个博客未来记录什么

## 一种简单写法

先写三段就够了：

1. 一个开场
2. 一件最近在意的事
3. 一个你想继续记录下去的理由

## 如果这是技术文章

也可以直接从一个小问题开始：

```js
function greet(name) {
  return `Hello, ${name}`;
}
```

最后给自己留一个小尾巴，比如下一篇想写什么。
